## Initiative Artifact Organization

When starting a new initiative, all artifacts go in their own folder using the format:
`initiatives/<INITIATIVE_ID>-<short-description>/`

Example: `initiatives/INI-001-cloud-migration/`

Each folder contains all initiative documents (brief, risk assessment, budget auth, legal clearance, execution plan, audit evidence, approvals).
