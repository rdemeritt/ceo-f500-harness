# /end-initiative — Initiative Close-Out

## Purpose

Formally closes an initiative, captures lessons learned, archives documentation, and updates all registers. Requires IA sign-off before close.

## Usage

```
/end-initiative [INI-YYYY-###]
```

---

## Close-Out Checklist

### 1. Delivery Confirmation
- [ ] All initiative deliverables confirmed complete
- [ ] Success metrics measured and documented (vs. original targets)
- [ ] Executive sponsor formally confirms completion
- [ ] Stakeholders notified of close

### 2. Financial Reconciliation
- [ ] Final budget vs. actual documented
- [ ] All invoices received and processed
- [ ] Any budget variance explained (>5% requires CFO sign-off)
- [ ] Ongoing operational costs captured in OpEx budget

### 3. IA Final Review (MANDATORY)
- [ ] IA reviews final initiative record
- [ ] All documents on file and complete
- [ ] No open audit findings
- [ ] Financial reconciliation validated
- [ ] Confirms initiative closed cleanly

### 4. Legal Close-Out
- [ ] All contracts fully executed (no outstanding signatures)
- [ ] IP transferred and documented (if applicable)
- [ ] No outstanding legal obligations
- [ ] GC confirms legal close

### 5. Lessons Learned
- What went well (keep doing)
- What didn't work (stop doing)
- What to improve (do differently)
- Key decisions made — were they right in hindsight?
- Process gaps identified

### 6. Archive
- Initiative record archived to corporate records system
- All documents tagged with retention period
- OKR contribution recorded
- Financial contribution recorded

```
INITIATIVE CLOSE-OUT REPORT
════════════════════════════════════════════════
Initiative: [INI-YYYY-###] — [Title]
Close Date: [ISO date]
Sponsor: [Role]
Initiative Lead: [Role]
════════════════════════════════════════════════

DELIVERY SUMMARY:
  Status:               [COMPLETE | PARTIAL | CANCELLED]
  Completion %:         [X%]
  Reason if partial:    [explanation]

FINANCIAL SUMMARY:
  Authorized Budget:    $[amount]
  Actual Spend:         $[amount]
  Variance:             $[amount] ([+/-X%])
  Variance Explanation: [if >5%]

SUCCESS METRICS:
  Metric 1: Target [X] → Actual [Y] → [MET | MISSED | EXCEEDED]
  Metric 2: Target [X] → Actual [Y] → [MET | MISSED | EXCEEDED]

IA SIGN-OFF:         [COMPLETE | PENDING]
GC SIGN-OFF:         [COMPLETE | N/A]

LESSONS LEARNED:
  Went well:       [2–3 items]
  Didn't work:     [2–3 items]
  To improve:      [2–3 items]

DOCUMENTS ARCHIVED: [Yes | No — remediation required]
════════════════════════════════════════════════
Initiative [INI-YYYY-###] is hereby CLOSED.
Closed by: [CEO | COO]
Date: [ISO date]
```
