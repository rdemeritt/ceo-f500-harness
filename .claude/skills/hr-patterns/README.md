# Skill: HR Patterns

## Purpose
Patterns for talent acquisition, performance management, compensation, and HR compliance.

## When to Invoke
- Designing hiring process or job requisition
- Conducting performance reviews
- Compensation analysis or benchmarking
- HR policy design or compliance check

## Hiring Process Pattern

```
REQUISITION → SOURCING → SCREENING → INTERVIEWS → OFFER → CLOSE

1. REQUISITION (CHRO + hiring manager)
   - Job description approved by CHRO
   - Headcount in approved plan
   - Compensation band set (CFO-approved ranges)
   - Level defined (IC / Manager / Director / VP / C-suite)

2. SOURCING
   - Internal posting first (5 business days minimum)
   - External sourcing if no qualified internal candidates
   - Recruiter engaged if >$150K role or specialized skill

3. SCREENING
   - Resume review against defined must-haves
   - Phone screen (30 min): background, motivation, compensation alignment
   - No questions about: age, family status, religion, national origin, health

4. INTERVIEWS (structured, same questions for all candidates)
   - Behavioral questions (STAR format: Situation, Task, Action, Result)
   - Skills assessment if technical role
   - Panel diversity (3+ interviewers, different backgrounds)

5. OFFER
   - Compensation within approved band
   - Offer letter from CHRO
   - Reference check before extending
   - Background check (GC advises on scope by jurisdiction)

6. CLOSE
   - Signed offer + start date
   - Onboarding plan assigned
   - IT/access provisioning (Day 1 ready)
```

## Compensation Benchmarking

Use two data sources minimum (e.g., Radford, Mercer, Levels.fyi for tech):
- Target: 50th–75th percentile for standard roles
- Target: 75th–90th percentile for critical/hard-to-fill roles
- Review benchmarks annually; adjust for market movement

## Performance Management Pattern

### Annual Review Cadence
- **Midyear check-in:** Progress vs. goals; development feedback
- **Year-end review:** Performance rating; compensation adjustment; goal setting

### Rating Scale (5-level)
| Rating | Definition |
|---|---|
| 5 — Exceptional | Significantly exceeds all expectations; top 5% of performers |
| 4 — Exceeds | Consistently exceeds most expectations |
| 3 — Meets | Fully meets all expectations; solid performer |
| 2 — Developing | Partially meets expectations; improvement needed |
| 1 — Below | Does not meet expectations; performance plan required |

### Performance Improvement Plan (PIP)
Required before termination for performance (not misconduct):
- Specific, measurable improvement targets
- Defined timeline (30/60/90 days typically)
- Support and resources provided
- Regular check-ins documented
- GC review before issuing and before any termination

## Employment Law Hard Rules
- Document everything — if it's not written, it didn't happen
- Terminations: always GC review if any claim risk
- Separation agreements: GC drafts; ADEA waiver requirements for 40+
- Non-competes: enforceability varies by state; GC must opine on each
